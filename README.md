Here are some key tips to get the most of what you learn from these files and any other sources you use:
- Concentration: Try reading in a calm area with no distractions.
- Separate Files: Put the codes of each lesson in one file and separate them to make things more tidy and easy to understand.
- Watch the Videos: Each file contains links to YouTube videos that explain the file (these videos are taken from Osama AL-Zero Python Course and they are in Arabic).
